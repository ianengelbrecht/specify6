Specify Network

  • For Mac: Choose Specify > Preferences > Sp Network.
  • For Windows and Linux: Choose Edit > Preferences > Sp Network

Specify Network Preference

  • About: Links to a web resource to learn more about the Specify Network service: https://www.specifysoftware.org/specify-network/
  • Register: Clicking this button will opt your collection in to the Specify Network.
  • Send Data: Once registered on the Specify Network site this button will send collection data for use in the listed services. There are two ways to send data, either all of your data or just an update. After initially sending all of the data in the database Updates will send only the data that has been added since the last upload.
