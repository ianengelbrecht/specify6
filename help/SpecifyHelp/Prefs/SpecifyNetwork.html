Specify Network

  • For Mac: Choose Specify > Preferences > Sp Network.
  • For Windows and Linux: Choose Edit > Preferences > Sp Network.

Specify Network Preference

 

  • Register: The Specify Network offers services in addition to checking specimen and taxon pages for those who are registered on the site.
  • About: Learn more about the Specify Network at this link to a web page.  
  • Send Data: Once registered on the Specify Network site this button will allow data to be sent for use in the additional services.
