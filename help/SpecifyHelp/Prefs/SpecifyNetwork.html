Specify Network

  • For Mac: Choose Specify > Preferences > Sp Network.
  • For Windows and Linux: Choose Edit > Preferences > Sp Network.

Specify Network Preference

 

  • Register: Clicking this button will opt your collection in to the Specify Network.
  • About: Links to a web resource to learn more about the Specify Network service.  
  • Send Data: Once registered on the Specify Network site this button will send collection data for use in the listed services.
